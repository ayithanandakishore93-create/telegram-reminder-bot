# Personal Jarvis Telegram Bot

A personal AI assistant for Telegram text only.

## Features
- Chat memory
- Personal memories
- Notes
- Tasks
- Natural language reminders
- Daily planning
- Review prompts
- SQLite persistence

## Commands
/start
/memory
/notes
/tasks
/reminders
/plan
/review
/delete <id>
/forget <id>
/clear

## Natural text examples
- remember my goal is build BLACKLEAF
- note: startup idea
- add task revise physics
- remind me tomorrow at 9am to call mom
- remind me after lunch to drink water
- plan my day
